# 📊 Project Overview
The objective of this project is to perform Exploratory Data Analysis (EDA) on a sales dataset to uncover patterns, trends, and business insights using Python and SQL.

The project also includes data visualization and dashboard mockup creation for better business decision-making.

---

# 🛠️ Tools & Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- SQL
- Jupyter Notebook
- Power BI / Excel

---

# 📂 Dataset Used
Superstore Sales Dataset

The dataset contains:
- Customer details
- Product information
- Sales
- Profit
- Quantity
- Region
- Category

---

# 🔍 Tasks Performed

## 1. Data Loading & Understanding
- Imported dataset using Pandas
- Checked dataset structure
- Analyzed column information

## 2. Data Analysis
Performed:
- Summary statistics
- Sales distribution analysis
- Category-wise analysis
- Region-wise sales analysis
- Profit analysis

## 3. Data Visualization
Created:
- Histograms
- Bar charts
- Scatter plots
- Heatmaps
- Line charts

## 4. SQL Analysis
Executed SQL queries to answer business questions such as:
- Top 5 products by sales
- Region-wise revenue
- Monthly sales trends
- Highest profit categories

## 5. Dashboard Mockup
Designed a static dashboard mockup displaying:
- Total Sales
- Total Profit
- Monthly Trends
- Category Performance
- Regional Analysis

---

# 📈 Key Insights
- Technology category generated the highest sales.
- Western region contributed maximum revenue.
- High discounts negatively affected profits.
- Sales and profit showed positive correlation.
