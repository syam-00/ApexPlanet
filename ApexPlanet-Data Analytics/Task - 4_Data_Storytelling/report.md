# Sales Dashboard Analysis

## Objective
The goal of this project is to analyze sales data to identify key drivers of revenue and profitability.

## Dataset
The dataset contains sales transactions including region, category, sales, profit, and quantity.

## Analysis Performed
- Sales performance analysis
- Regional sales comparison
- Category profit analysis

## Key Insights
- Technology products generate the highest profit.
- Furniture shows lower profitability in some regions.
- West region contributes significantly to total revenue.

## Hypothesis Testing
Hypothesis:
Technology category generates higher profit than Furniture.

Test Used:
Two-sample T-test

Result:
If p-value < 0.05 → Technology category significantly outperforms Furniture.

## Conclusion
Businesses should focus on high-performing product categories and optimize underperforming segments.
